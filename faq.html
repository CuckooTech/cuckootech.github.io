FAQ

Q:I stuck the sharing code, but did not show the video. Is it a malfunction?

A:If this happens, don't worry. Because this is a blockchain application, it will be slow to start as a new user, and it will be smooth after a few days. We recommend watching some short videos first.

Q:Why can't my sharing code be pasted?

A:Please do not worry, copy the sharing code you want to share, then click the button above.

Q:If I change a computer, can I still log in to the original computer account?

Sure. You only need to back up the original account, then import it into the new computer.
The following steps are:
I. Click the “system settings” button.
II. Click the "Back-up" button to create the account backup file.
III. Install “Cuckoo” on the new computer, click "Recovery" button, then import the files which were backed up before.
